# Sleeper
Simple timer. Written by c++

# Usage
`sleeper -command -time -log`
You have to add 3 parameters for sleeper run.
### Example
`sleeper -m 15 -t`

# Argvs
PREFIX:       `-`
### Command
SECOND:       `-s`\
MILLISECOND:  `-ms`\
HOUR:         `-h`\
MINUTE:       `-m`
### Log
LOG TRUE:     `-t`\
LOG FALSE:    `-f`
